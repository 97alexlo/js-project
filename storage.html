Window.localStorage

  • localStorage is one of the two mechanisms of the Web Storage API
  • It allows JavaScript websites to save key-value pairs (both as strings) in a browser's memory even after the page is closed
  • In google chrome and firefox, localStorage is stored in an SQLite file in the user's profile folder
  • It is a read-only property window interface thats allowas the user to access a Storage object which can be used to access the document's origin local storage space
